Portfolio Website is a fully responsive web created with Flask framework, Boostrap, HTML, CSS and JavaScript. This is a personal website with the intention of acting as a source of information about my professional skills.

## Features

- Information about me, my professional work and ways to contact.
- Email form to receive messages from visitors in mailbox.
- Links to my professional profiles in Github and Linkedin.
- Option to download or visualice my resume.

## Deployment
This project is deployed on AWS through Zeet, and can be accessed at https://adriantarin.com.
